## ytgrab core: the resilient downloader

- ask: ytgrab, end to end, click-to-run (your settled option 1) — you said you would not do the terminal yourself, so there is none to do. A "ytgrab" entry is in your applications menu; clicking it the first time asks for the channel and the save folder with two dialogs (no file editing), then opens a terminal window showing the download running; clicking it again any time resumes — a lock means a second click never double-runs, and if a run is already live it just tells you. Underneath, the core (`ytgrab.py`) wraps yt-dlp: a whole channel oldest-first, ≤720p merged to mp4, exact resume across any stop/restart/crash (download-archive + partial continue), a polite randomised sleep so it is never rate-limited, and a free-space guard that stops with a message instead of crashing (a click resumes once space is freed). Two build slices — the core and the desktop shortcut + launcher — both built; this one card accepts the whole click-to-run experience.
